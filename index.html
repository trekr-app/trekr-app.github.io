Desc

This is some warm-welcoming filler text that hopes to introduce potential users to Trekr. Trekr is an android app that allows users to map trails and store them on their device for later use. Trekr is intended to help people who are unfamiliar with an area become knowledgeable about the trails in the region. In addition to this, Trekr can help a lost user find their way to civilization and provide a number of means to keep a user safe. Some interesting features include the use of the Emergency mode that will display a user's latitude and longitude as well as provide the option for a user to send a pre-typed SMS to a specific contact in the case of an emergency.
